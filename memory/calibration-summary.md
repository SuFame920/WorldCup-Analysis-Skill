| 让球 | 2/4 | **四场全押受让方**:favorite被逼平的两场✓(BEL/URU),favorite赢透的两场✗(ESP/EGY) |

实际比分:Spain 4-0 / Belgium 0-0 / Uruguay 2-2 / Egypt 3-1。**4 场 10 球,3 场打出大球。**

### 根因分析(Why?,5-whys)
1. 为什么错?→ 四场我都偏**小球 + 受让方 + favorite小净胜球**。
2. 为什么每场都这样?→ 用了**同一套叙事模板**("强权创造多但临门差 + 对手摆大巴 → 低分、险胜、平局风险")套到每一场,不论对象差异。=技能明确警告的"为任何结果都能编出顺畅故事"的失败模式。
3. 为什么是这套模板?→ 两组 **MD1 全部低分闷平**(0-0/1-1/1-1/2-2),我把**1 个轮次的小样本**外推成"这两组就是低分龟缩组"。=近因过度外推。
4. 为什么近因压过了市场锚?→ 我把自己的定性"机制"当成比 **盘口 + xG 基准率**更强的证据,在 totals/让球 上**反复同向偏离市场**,而每次偏离没有被独立验证。
5. 为什么过度相信故事?→ 结构缺陷:技能奖励"机制驱动的调整",但我**缺少组合级偏差自检**(没发现自己在多场相关比赛里做了同方向调整),且 totals/让球 的调整**帽子不够紧**。

### 重要的反面教训(别过度纠偏)
- 平局直觉是**对**的:BEL、URU 都平,我的**不败(1X)双选两场都中**;BEL-IRR 是**过程胜场**(让球✓大小✓)。所以修正**不是**"一律改大球/一律追favorite"。
- 会话内我曾因 Spain 一场就想把 Uruguay 翻成"乌拉圭过盘/大球":结果**大球对、但"乌拉圭过盘"会把原本✓的受让方翻成✗**。→ **别凭单场结果反向追逐**;要修的是系统性偏差,不是上一场的方向。

### Standing corrections(下次预测载入为弱先验)
1. **杀掉系统性 under-bias。** 当**双方都展示过得分点**且**赛况激励追分**(出线缠斗/必须赢)时,totals 默认**贴市场或偏中性**,不要反射性押小球。我的小球倾向需-1 档(更靠近盘口)。[magnitude: small-moderate]
2. **别给 favorite 净胜球封顶太低。** 强队(尤其有精英终结者)对弱旅/低位,要给 3-0/4-0、3-1 这类更高净胜球**更多矩阵质量**;比分别只收在 1-0/2-0。[small-moderate]
3. **"哑火强队"分型(关键)。** 精英终结者的**短期冷却**(如西班牙 Yamal/Oyarzabal)→ **会回归**,别反买其让球/大球;**结构性哑火**(如比利时:老化/无稳定运动战终结点/缺核心)→ **会延续**,可继续看低进球/受让方。判别看:终结者个体质量、阵容是否齐整、哑火是几场还是长期。[moderate]
4. **"低位防守 ≠ 低进球"。** 摆大巴弱旅也会进球(世界波/反击/定位球/对手失误);佛得角已两平强队还进 3 球。别默认其零进球。[small-moderate]
5. **受让方 value 不能只靠"赢不透"的故事。** 要对冲对手的质量兑现(萨拉赫级别球员会在均势局后段拉开)。同向受让方的多场押注要做组合自检。[small-moderate]
6. **组合级偏差自检(流程性)。** 一个批次里如果发现自己在多场里做了**同方向**的 totals/让球 调整,停下来问:这是各场独立证据,还是同一套故事的复制?后者就向市场收敛。[process]

### 仍然有效、别误伤的东西
- 平局风险识别 + **不败(1X)双选**作为对强队的稳妥对冲(本批次救回两场)。
- 让球受让方在**真正势均力敌或favorite结构性疲软**时仍有价值(BEL/URU 验证)。
- 1X2 方向判断整体可用(2 胜 2 平,且两平我都标了高平局风险)。

### 校准闭环验证(核心结论:反思机制有效)
- **standing correction #2(别给 favorite 净胜球封顶太低)被强力验证:** 上批次比分 0/4,本批次仅靠"把强队比分上移到 2-0/3-0/2-1"就拿下 3/4。**反思 → Channel H → 下次预测**的闭环**确实加了信号,不是讲故事。** → #2 加权,保留为强先验。
- **standing correction #1(杀 under-bias)总体有效但出现 overcorrection:** under-bias 已消除(大小球错向变双向)。但 Argentina 一场把"杀 under"误用成"一律 Over"(精英绞杀被动弱旅本是低分场,实 2-0)→ 验证了 reflection-protocol 第 4 步的反面教训:**别为纠一个错而矫枉过正。**

### 根因分析(Why?,5-whys,针对残留偏差)
1. 残留错在哪?→ 大小球 2/4、半全场 2/4。
2. 大小球为什么错?→ **场景分型用反了**:开放生死局(Alg)押了小、精英绞杀局(Arg)押了大——两场都把 game-state 映射搞反。
3. 为什么映射反?→ 我有 #1 这条"偏大"修正,但**没有明确的判别规则**告诉我"何时偏大 / 何时仍可偏小"。修正是个标量(偏大),不是个条件函数。
4. 半全场为什么两场都错 HT?→ 我有一个**过强的"半场平"默认**(谨慎开局先验),但本批次有早段终结点的 favorite(哈兰德/约旦先进)上半场就破僵。
5. 结构缺什么?→ 缺**"场景分型 → totals/HT"的条件映射表**;totals 修正写成了无条件标量,半全场缺"早段进球能力 → 半场领先概率上调"的钩子。

- **两个大 miss(Por 5-0、Eng 0-0)主因是 finishing variance vs xG,不是过程全错(诚信红线:不许全记成模型错而矫枉过正):** Por xG 2.43→5 球(高效终结+乌龙+替补补刀);Eng xG 1.28+78.8% 控球+19 射→0 球(终结差+阿萨雷神扑+缺开锁手)。**两场"球队表现/方向"≈对,偏差在比分。**
